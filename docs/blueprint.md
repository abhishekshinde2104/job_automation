# **App Name**: Resume Tailor AI

## Core Features:

- Document Upload: Allow users to upload their resume, cover letter, and the job description.
- AI-Powered Tailoring Tool: Use an AI tool to analyze the job description and tailor the resume and cover letter by re-wording existing descriptions or creating new ones to highlight the qualifications requested by the job posting.
- Download Tailored Documents: Allow users to download the tailored resume and cover letter.

## Style Guidelines:

- Primary color: Soft blue (#77B5FE) to convey trust and professionalism.
- Background color: Light gray (#F5F5F5) for a clean and modern look.
- Accent color: Teal (#008080) for call-to-action buttons.
- Body and headline font: 'Inter', a grotesque-style sans-serif.
- Use professional and clear icons for document upload and download actions.
- Maintain a clean and organized layout to ensure readability and easy navigation.
- Use subtle animations for feedback on document upload and processing.
